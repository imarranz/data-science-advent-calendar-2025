Axis transformations can reveal hidden patterns in your data. In Matplotlib, you can switch between linear, logarithmic, and symmetric log scales â€” or even invert axes entirely â€” with just a few lines.

Basic example


import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(1, 100, 200)
y = x ** 2

fig, ax = plt.subplots()
ax.plot(x, y, label="y = xÂ²")

ax.set_xscale("linear")
ax.set_yscale("log")     # logarithmic y-axis
ax.set_xlabel("x")
ax.set_ylabel("y (log scale)")
ax.set_title("Log Scales and Axis Tricks")
ax.legend(frameon=False)

plt.tight_layout()
plt.show()

The set_yscale("log") call compresses large values and spreads small ones, making exponential or power-law relationships easier to visualize.

Switching scales


ax.set_xscale("log")     # both axes log
ax.set_yscale("symlog")  # symmetric log: handles negatives
ax.set_yscale("logit")   # for proportions (0â€“1)

"symlog" adds a linear region around zero, while "logit" is handy for probabilities.

Inverting axes


ax.invert_xaxis()   # flip leftâ†”right
ax.invert_yaxis()   # flip upâ†”down

Axis inversion is useful for geological or imaging data, where the origin is traditionally at the top.

Fine control


ax.set_xlim(1, 100)
ax.set_ylim(1e0, 1e5)
ax.grid(True, which="both", ls="--", lw=0.5)
ax.minorticks_on()   # minor ticks on log scales

The combination of gridlines and minor ticks is especially helpful when working with log scales â€” it gives visual anchors without overwhelming the viewer.
