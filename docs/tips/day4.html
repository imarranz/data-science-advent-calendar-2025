Missing values (NaN = ‚ÄúNot a Number‚Äù) are inevitable in real-world data. pandas provides simple and powerful tools to detect, remove, or replace them.

Detecting missing values


import pandas as pd
import numpy as np

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", None],
    "age": [25, np.nan, 30, 40],
    "score": [85, 92, None, 70]
})

print(df)


# Detect which cells are missing
df.isna()

# Count missing values per column
df.isna().sum()

# Quick overview
df.info()

isna() returns a Boolean mask indicating where values are missing. You can combine it with sum() or any() to summarize missingness.

Dropping missing data


# Drop any rows containing NaN
df_clean = df.dropna()

# Drop only if all values in a row are NaN
df_partial = df.dropna(how="all")

# Drop columns with missing data
df_cols = df.dropna(axis="columns")

Dropping rows can be convenient, but it‚Äôs risky if you lose too much data. In practice, it‚Äôs often better to fill or impute missing values.

Filling missing data


# Replace NaN with a constant
df_filled = df.fillna(0)

# Fill with a statistic (per column)
df_filled_mean = df.fillna(df.mean(numeric_only=True))

# Forward-fill (use last valid value)
df_ffill = df.fillna(method="ffill")

# Backward-fill (use next valid value)
df_bfill = df.fillna(method="bfill")

You can fill selectively per column too: 


df["age"] = df["age"].fillna(df["age"].mean())
df["name"] = df["name"].fillna("Unknown")

Practical tip üß†

Always check df.isna().sum() before deciding how to handle missing data. Dropping is fast but potentially destructive. Imputing (filling) preserves structure and is usually safer ‚Äî especially before modeling.
