A well-placed legend turns a good plot into a clear one. In Matplotlib, the legend can be styled and positioned precisely â€” just like any other plot element.

Basic example


import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y1 = np.sin(x)
y2 = np.cos(x)

fig, ax = plt.subplots()
ax.plot(x, y1, label="sin(x)")
ax.plot(x, y2, label="cos(x)")

ax.legend(frameon=False, title="Functions",
          bbox_to_anchor=(1.02, 1), loc="upper left")

ax.set_xlabel("x")
ax.set_ylabel("value")
ax.set_title("Elegant Legend Example")

plt.tight_layout()
plt.show()

This places the legend outside the plot area, aligned to the upper left, without a frame. The bbox_to_anchor argument controls the exact anchor point (here, slightly outside the axes).

Styling options


# Legend with frame and custom style
ax.legend(
    title="Series",
    frameon=True,
    facecolor="white",
    edgecolor="gray",
    fontsize=10,
    title_fontsize=11,
    loc="upper right"
)

You can change font sizes, colors, alignment, and more. Setting frameon=False removes the box, while ncol=2 spreads items across columns. 


ax.legend(ncol=2, loc="lower center", bbox_to_anchor=(0.5, -0.2))

Multiple lines and automatic labels


# Automatically use labels from plotted lines
for freq in [1, 2, 3]:
    ax.plot(x, np.sin(freq * x), label=f"{freq} Hz")

ax.legend(title="Frequency", ncol=3, frameon=False)

Matplotlib collects all lines with a label argument and includes them automatically. To skip a line in the legend, set label="_nolegend_".
