When you need to compare how values are distributed across categories, box plots and violin plots are your best friends. They show medians, spread, and symmetry at a glance â€” revealing patterns that simple averages can hide.

Box plot â€” the statistical summary


import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

# Example dataset
np.random.seed(42)
df = pd.DataFrame({
    "group": np.repeat(["A", "B", "C"], 100),
    "value": np.concatenate([
        np.random.normal(0, 1, 100),
        np.random.normal(1, 1.2, 100),
        np.random.normal(2, 0.8, 100)
    ])
})

sns.set_theme(style="whitegrid")
sns.boxplot(data=df, x="group", y="value", palette="Set2")

plt.title("Box Plot â€” Distribution Summary")
plt.show()

A box plot shows the median (line), interquartile range (box), and potential outliers (points beyond the whiskers). Itâ€™s ideal for comparing groups quickly.

Violin plot â€” distribution shape


sns.violinplot(data=df, x="group", y="value", inner="quartile", cut=0, palette="Set3")
plt.title("Violin Plot â€” Distribution Shape")
plt.show()

A violin plot adds a kernel density estimate on each side of the box, showing whether the distribution is symmetric, skewed, or multimodal. The cut=0 parameter prevents the violin from extending beyond the observed data range.

Combine and compare


# Combine both for richer insight
sns.violinplot(data=df, x="group", y="value", color="0.9", inner=None)
sns.boxplot(data=df, x="group", y="value", width=0.2, showcaps=False, boxprops={"zorder": 2})
plt.title("Box + Violin Combined")
plt.show()

Overlaying box plots on violins can give you both precision and shape â€” just be sure to tone down colors for readability.

Orientation and grouping


# Horizontal orientation
sns.boxplot(data=df, y="group", x="value", orient="h")

# Multiple subgroups
df["condition"] = np.random.choice(["X", "Y"], size=len(df))
sns.boxplot(data=df, x="group", y="value", hue="condition", palette="muted")

Orientation can make comparisons easier for long labels, and using hue reveals differences between subgroups in the same figure.
