Set a global theme


import seaborn as sns
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

# Global style for all plots
sns.set_theme(style="whitegrid", palette="deep")

# Example dataset
x = np.linspace(0, 10, 100)
df = pd.DataFrame({"x": x, "sin": np.sin(x), "cos": np.cos(x)})

sns.lineplot(data=df, x="x", y="sin", label="sin(x)")
sns.lineplot(data=df, x="x", y="cos", label="cos(x)")
plt.title("Seaborn Style & Palette")
plt.show()

The theme controls the overall background, grid, and axes style. The palette defines the color cycle used for lines, bars, or points.

Available styles


for style in ["white", "dark", "whitegrid", "darkgrid", "ticks"]:
    sns.set_style(style)
    sns.lineplot(data=df, x="x", y="sin")
    plt.title(style)
    plt.show()

whitegrid is popular for clarity on numeric data, while dark or ticks gives a more minimalist tone.

Color palettes


# Named palettes
sns.set_palette("pastel")
sns.set_palette("muted")
sns.set_palette("colorblind")  # accessible palette

# Custom palettes from color maps
palette = sns.color_palette("viridis", as_cmap=True)
sns.heatmap(np.random.rand(10, 10), cmap=palette)

Palettes can be categorical (discrete colors) or continuous (color gradients). You can also build your own with sns.color_palette() or use any Matplotlib colormap.

Context: adjust scale globally


sns.set_context("talk")      # larger fonts for presentations
sns.set_context("paper")     # smaller fonts for compact figures

Context scales fonts, line widths, and markers for different output formats â€” essential when reusing the same code for papers, slides, or dashboards.
