Once your plot looks right on screen, the final step is saving it cleanly for reports, papers, or presentations. Matplotlib lets you control figure size, resolution, and format precisely â€” so your visuals stay sharp anywhere.

Basic high-resolution export


import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y = np.sin(x) * np.exp(-0.1 * x)

fig, ax = plt.subplots(figsize=(5, 3))  # size in inches (width, height)
ax.plot(x, y, color="steelblue", lw=2)
ax.set(xlabel="x", ylabel="y", title="High-Resolution Export")

# Save at 300 DPI (print quality)
fig.savefig("plot.png", dpi=300, bbox_inches="tight")

dpi (dots per inch) controls sharpness â€” use 300+ for print, 150 for slides, and 100 for quick drafts. bbox_inches="tight" trims extra whitespace around the figure.

Save in different formats


fig.savefig("plot.pdf")      # vector format, best for print
fig.savefig("plot.svg")      # scalable web graphic
fig.savefig("plot.jpg", quality=95)  # lossy but smaller
fig.savefig("plot.tiff", dpi=600)    # for journal submissions

Use vector formats (.pdf, .svg) for line art and text â€” they scale perfectly. Use raster formats (.png, .jpg, .tiff) for photos or dense plots.

Programmatic control of size and layout


fig.set_size_inches(6, 4)
fig.tight_layout(pad=0.2)

# or directly in savefig (rare but handy)
fig.savefig("plot.png", dpi=300, bbox_inches="tight", pad_inches=0.1)

Adjusting size in inches ensures consistent scaling across formats. tight_layout() and pad_inches help balance spacing around labels and titles.

Transparent and background options


# Transparent background (great for slides)
fig.savefig("plot_transparent.png", dpi=300, transparent=True)

# White background enforced
fig.savefig("plot_whitebg.png", facecolor="white")

Transparency works best in PNG and SVG formats. Always test legibility on both light and dark backgrounds before sharing.

Batch saving multiple figures


for i, freq in enumerate([1, 2, 3], start=1):
    fig, ax = plt.subplots()
    ax.plot(x, np.sin(freq * x))
    fig.savefig(f"plot_{i}.png", dpi=200, bbox_inches="tight")
    plt.close(fig)   # free memory
