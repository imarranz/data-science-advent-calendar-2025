A scatter plot with a regression line is one of the quickest ways to explore relationships between variables. Seaborn makes it easy to visualize both the trend and its uncertainty in a single command.

Basic regression plot


import seaborn as sns
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

# Example data
np.random.seed(0)
x = np.linspace(0, 10, 100)
y = 2 * x + 1 + np.random.randn(100) * 2
df = pd.DataFrame({"x": x, "y": y})

sns.regplot(data=df, x="x", y="y", ci=95,
            scatter_kws={"s": 30, "alpha": 0.7, "color": "steelblue"},
            line_kws={"color": "darkorange", "lw": 2})

plt.title("Scatter + Regression with Confidence Band")
plt.show()

The regression line shows the estimated linear trend; the shaded region represents a 95% confidence interval around that fit.

Customize the model and fit


# Disable confidence interval
sns.regplot(data=df, x="x", y="y", ci=None)

# Use a polynomial fit instead of linear
sns.regplot(data=df, x="x", y="y", order=2)

# Log-transform the x-axis before fitting
sns.regplot(data=df, x=np.log(df["x"] + 1), y="y")

Seaborn uses scipy.stats.linregress under the hood for simple linear regression. For more complex models (like logistic or nonparametric fits), use sns.lmplot() or sns.regplot(..., logistic=True).

Multiple groups


# Multiple categories â†’ use hue for grouped regression
df["group"] = np.random.choice(["A", "B"], size=len(df))

sns.lmplot(data=df, x="x", y="y", hue="group",
           palette="Set2", height=4, aspect=1.2)

Each group gets its own regression line and confidence band. lmplot() creates a new figure and supports grouping, faceting, and easy layout.
