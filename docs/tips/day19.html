Basic labeled plot


import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y = np.sin(x) * np.exp(-0.1 * x)

fig, ax = plt.subplots()
ax.plot(x, y, color="steelblue", lw=2)

# Axis labels and title
ax.set(
    xlabel="Time (s)",
    ylabel="Amplitude",
    title="Damped Sine Wave"
)

fig.tight_layout()
plt.show()

ax.set() is a shorthand for setting multiple attributes at once. tight_layout() prevents titles and labels from being cropped.

Adding annotations


# Annotate a local maximum
x0 = x[np.argmax(y)]
y0 = y.max()

ax.annotate(
    "Peak",
    xy=(x0, y0),             # point to annotate
    xytext=(x0 + 1, y0 + 0.2),  # label position
    arrowprops=dict(arrowstyle="->", lw=1.2, color="gray"),
    fontsize=10,
    ha="left"
)

annotate() places text relative to a data point and can draw arrows between the two. Arrows and alignment (ha, va) keep labels readable and professional.

Multiple text styles


# Inline text
ax.text(2, 0.5, "Transient phase", fontsize=9, color="darkred")

# Figure-level title
fig.suptitle("Experiment A â€” Signal decay", fontsize=13, fontweight="bold")

Use ax.text() for free placement and fig.suptitle() for an overall title above all subplots.
