Quick EDA (Exploratory Data Analysis) means getting signal fast: shapes, types, missingness, basic stats, and a feel for pairwise relationships. This combo gives you a dependable first pass in seconds.

Structural overview


# Dimensions, column names, dtypes, and non-null counts
print(df.shape)     # (rows, columns)
print(df.columns)   # Index([...])
df.info()           # dtypes + non-null counts (great for missingness)

Descriptive statistics


# Numeric & categorical summary
summary = df.describe(include="all", datetime_is_numeric=True)
print(summary)

describe() gives count/mean/std for numeric columns and freq/top for categoricals. If you have datetimes, datetime_is_numeric=True treats them as numbers (useful for ranges).

Quick distribution check


import seaborn as sns
import matplotlib.pyplot as plt

num = df.select_dtypes("number")

# Histograms for all numeric columns (compact loop)
for col in num.columns:
    sns.histplot(num[col], kde=True)
    plt.title(f"Distribution ‚Äî {col}")
    plt.show()

Correlation sneak peek


# Pairwise correlations (numeric only)
corr = num.corr(numeric_only=True)  # on recent pandas, 'numeric_only' is implicit
print(corr.round(3))

# Optional heatmap
sns.heatmap(corr, annot=False, cmap="viridis", center=0)
plt.title("Correlation (numeric)")
plt.tight_layout()
plt.show()

Pairwise relationships at a glance


# Pairplot: small multiples for scatter + hist
# 'corner=True' shows only the lower triangle (cleaner);
# 'plot_kws' keeps points readable
sns.pairplot(num, corner=True, diag_kind="hist",
             plot_kws={"alpha": 0.6, "s": 25, "edgecolor": "none"})
plt.show()

For mixed data with categories, coloring by a grouping variable can reveal separations: 


# If you have a categorical "target" or "group"
if "target" in df.columns:
    sns.pairplot(df.select_dtypes("number").assign(target=df["target"]),
                 hue="target", corner=True, diag_kind="hist",
                 plot_kws={"alpha": 0.6, "s": 25})
    plt.show()

Performance & practicality üß†

  • Large data? Sample first: df.sample(5000, random_state=0) before pairplot.
  • High-cardinality categoricals? Use value_counts() on top suspects to understand tail behavior.
  • Dtypes matter: fix integers stored as strings and parse dates early (pd.to_datetime).
  • Memory check: df.memory_usage(deep=True).sum() to see if you need downcasting.
