In pandas, understanding the difference between a Series and a DataFrame is essential. A Series is a one-dimensional labeled array, while a DataFrame is a two-dimensional table made up of multiple Series, all aligned by their index.

You can think of a Series as a single column in Excel, and a DataFrame as the entire spreadsheet â€” a collection of columns that share the same row labels. 


import pandas as pd

# A Series: one labeled column
s = pd.Series([10, 20, 30], name="values")
print("Series:")
print(s)
print("Dimensions:", s.ndim, "| Shape:", s.shape)

# A DataFrame: multiple columns aligned by index
df = pd.DataFrame({
    "A": [10, 20, 30],
    "B": [1.1, 2.2, 3.3],
    "C": ["x", "y", "z"]
})
print("\nDataFrame:")
print(df)
print("Dimensions:", df.ndim, "| Shape:", df.shape)

Notice that the Series is one-dimensional (ndim = 1), whereas the DataFrame is two-dimensional (ndim = 2). Internally, each column of a DataFrame is a Series sharing the same index.
