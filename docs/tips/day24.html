Multiple plots in one figure help compare patterns side by side â€” but without discipline, they quickly become unreadable. Matplotlib gives fine control over shared axes, spacing, and alignment for clean, professional layouts.

Basic shared-axis layout


import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y = np.sin(x)

fig, axes = plt.subplots(2, 2, sharex=True, sharey=True, figsize=(6, 4))

for i, ax in enumerate(axes.ravel(), start=1):
    ax.plot(x, y * i, lw=2)
    ax.set_title(f"Plot {i}", fontsize=9)

fig.suptitle("Clean Multi-Subplot Layouts", fontsize=12, fontweight="bold")
fig.tight_layout()
plt.show()

sharex=True and sharey=True align axes perfectly across subplots, while tight_layout() ensures spacing adjusts automatically to avoid overlap.

Adjust spacing manually


# Add space between rows or columns
fig.subplots_adjust(hspace=0.3, wspace=0.25)

hspace and wspace (height and width spacing) are fractions of the subplot dimensions. Use them when tight_layout() still leaves labels too close.

Shared labels and global titles


# Common labels for all subplots
fig.text(0.5, 0.02, "X-axis label", ha="center")
fig.text(0.04, 0.5, "Y-axis label", va="center", rotation="vertical")

# Figure-level title
fig.suptitle("Experiment Overview", y=1.02, fontsize=13)

Using gridspec for flexible layouts


import matplotlib.gridspec as gridspec

fig = plt.figure(figsize=(6, 4))
gs = gridspec.GridSpec(2, 3, figure=fig)

ax1 = fig.add_subplot(gs[0, :])      # top row spans all columns
ax2 = fig.add_subplot(gs[1, 0:2])    # bottom left (2 columns)
ax3 = fig.add_subplot(gs[1, 2])      # bottom right

ax1.plot(x, np.sin(x)); ax2.plot(x, np.cos(x)); ax3.plot(x, np.tan(x))
fig.tight_layout()
plt.show()

GridSpec gives surgical control â€” perfect for dashboards, insets, or asymmetrical designs.

Aligning labels and legends


for ax in axes[-1, :]:
    ax.set_xlabel("Time (s)")
for ax in axes[:, 0]:
    ax.set_ylabel("Value")

# Align all y-labels
fig.align_ylabels(axes[:, 0])

Consistent labeling across subplots keeps the grid visually balanced. align_ylabels() and align_xlabels() are your friends for that last bit of polish.

Complex grids: Subgrids and subfigures


fig = plt.figure(constrained_layout=True, figsize=(7, 4))
subfigs = fig.subfigures(1, 2, wspace=0.07)

axs_left = subfigs[0].subplots(2, 1, sharex=True)
axs_right = subfigs[1].subplots(2, 1, sharex=True)

for ax in axs_left.ravel():
    ax.plot(x, np.sin(x))
for ax in axs_right.ravel():
    ax.plot(x, np.cos(x))

subfigs[0].suptitle("Sine")
subfigs[1].suptitle("Cosine")
fig.suptitle("Subfigures Example", fontsize=12)
plt.show()
